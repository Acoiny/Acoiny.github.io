Automatic image compressing

I once had to upload many images to a wordpress site that only allowed files of up to 2Mb to be uploaded. Because the images I had to upload were all way bigger, I wrote a python script to compress many images to a certain size. It can be found here.

OTH scrape tool

I wrote a simple web scraper, that fetches the blackboard ot the OTH Regensburg's IM Faculty. It can also fetch the Mensa menu and display it either directly in the terminal or saves it to a markdown file.

TicTacToe clone?
